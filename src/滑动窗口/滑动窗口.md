# 滑动窗口求窗口最大值
```
// 维持一个单调的双头队列即可
Deque<Integer> deque = new LinkedList<>();
public void push_back(int value) {
    while (!deque.isEmpty() && deque.peekLast() <= value) {
        deque.pollLast();
    }
    deque.addLast(value);
}
```
使用此思路可A：
